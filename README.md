# Laboratorio-practico-1-Venta-de-Autos
Trabajo practicó de las clases del Informatorio-Chaco, utilizado un dataset público.

Analisis de un dataset publico https://www.kaggle.com/datasets/suraj520/car-sales-data , se realizo un análisis exploratorio de datos (EDA) a las ventas de una tienda de vehiculos.

Pasos realizados:

Extracción:     *Se importó el dataset desde una fuente pública kaggle.

Transformación:
                *Eliminación de nulos y duplicados.
                *Conversión de la columna de fecha a tipo datetime.

Analisis:
                Analisis de Ventas realizadas
                Analisis de Ingresos de los Vendedores
